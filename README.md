# starself

Astrology Calc — birth chart generator for web, iOS, and Android.

Built with [Expo](https://expo.dev) (React Native + TypeScript), targeting all three platforms from one codebase.

## Development

```
npm install
npm run web      # run in browser
npm run ios      # run in iOS simulator
npm run android  # run in Android emulator
```

To preview on your phone, install the [Expo Go](https://expo.dev/go) app and run `npx expo start`, then scan the QR code.
